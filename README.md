# ate

a text editor in pure assembly

--- 

ate is a pure text editor which only uses assembly as its languages. This is meant to be a fun project, and is in no way meant to replace well - known IDEs or code editors like Vim.
This is meant for people who only wish for a super lightweight, small and opinionated text editor.
